# ZIVA — Data Wrangling & Exploratory Data Analysis

Repository ini berisi kode analisis data wrangling dan eksplorasi data (EDA) yang mendukung esai "ZIVA: Explainable AI untuk Transformasi Seleksi Embrio IVF Menuju Indonesia Sehat 2045" pada Statistics Essay Competition Satria Data 2026, Universitas Diponegoro.

## Sumber Data
Dataset yang digunakan adalah IVFNet, dataset publik yang memuat citra embrio time-lapse dan data klinis tabular terkait keberhasilan implantasi IVF.

## Cara Reproduksi
1. Clone repository ini
2. Install dependencies: pip install -r requirements.txt
3. Unduh dataset IVFNet dari sumber resmi
4. Jalankan notebook di folder notebooks/

## Catatan
Analisis ini merupakan studi kelayakan terhadap karakteristik teknis dataset publik, bukan validasi performa model ZIVA secara penuh.
